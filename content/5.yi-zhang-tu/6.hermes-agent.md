# Hermes Agent 技术架构

自主规划 · 工具调用 · 长期记忆 · 安全可控的通用智能体框架。

---

![Hermes Agent 技术架构](/hermes-agent-architecture.png)

### 架构分层

| 层级 | 说明 |
|------|------|
| **应用场景层** | 智能客服、问答、数据分析、办公自动化、软件开发、运维监控、个人助手、行业定制 |
| **接入层** | Web UI、移动端、API/SDK、CLI、第三方平台 |
| **智能体核心层** | 感知、规划推理、行动、记忆、状态管理、安全控制 |
| **工具与服务层** | 内置工具、MCP 工具生态、API 服务、连接器、插件市场 |
| **资源层** | 知识库、向量数据库、关系型数据库、文件存储、缓存、消息队列 |
| **模型层** | LLM（GPT-4、Qwen、Llama 等）、Embedding 模型、语音模型 |
| **基础设施层** | 容器部署、云平台、微服务、监控、CI/CD、安全 |
