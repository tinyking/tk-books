# GitHub

全球最广泛采用的 AI 驱动开发者平台。

---

![GitHub](/github-homepage.png)

### 核心功能

- **代码托管**：支持 Git 版本控制，提供无限的公开和私有仓库
- **Pull Requests**：代码审查与协作合并的工作流
- **GitHub Copilot**：AI 驱动的代码补全与生成
- **Actions**：CI/CD 持续集成与持续部署
- **GitHub Marketplace**：丰富的开发者工具生态

### 开源生态

热门开源项目示例：

- `microsoft/TypeScript` — TypeScript 编程语言
- `vercel/next.js` — React 服务端渲染框架
- `microsoft/vscode` — 开源代码编辑器
- `tailwindlabs/tailwindcss` — 实用优先的 CSS 框架
- `shadcn/ui` — 美观的 UI 组件库
- `facebook/react` — 用于构建用户界面的 JavaScript 库
